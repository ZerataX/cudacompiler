inlinestatic

Creates a new PaddingArg with the passed ArrayArg interpreting the array as matrix.
When uploaded, the PaddingArg create a new array (interpreting as matrix) on the device with passed values for dimensions using the values from ArrayArg and paddingValue to fill up remaining values. if isDownload is true: after kernellaunch the result will be downloaded to the passed ArrayArg (without the padding).
Only supports arrays with elementsize 1, 2 or 4 bytes.
