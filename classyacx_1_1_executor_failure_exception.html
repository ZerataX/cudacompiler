Detailed Description

Exception class for Errors occur during execution of a kernel.
Especially errors while compiling kernel with nvrtc or while using CUDA driver api create this Exception.
Errors like segmentation faults (e.g. passing invalid arguments to CUDA kernel) are not caught and will crash the execution of the running program.
