Detailed Description

Class representing an int-array-argument for the kernel.
Note: The size in bytes for one int-element is fixed and may differ from the size of the corresponding data-type in CUDA, which is depending on your system. So make sure the size of the corresponding data-type is matching SIZE_BYTES to avoid unexpected segmentation faults.
