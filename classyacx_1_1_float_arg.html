Detailed Description

Class representing an float-array-argument for the kernel.
Note: The size in bytes for one float-element is fixed and may differ from the size of the corresponding data-type in CUDA, which is depending on your system. So make sure the size of the corresponding data-type is matching SIZE_BYTES to avoid unexpected segmentation faults.
